CSV Data Sorting & Searching Tool (Python + Pandas)

This is a lightweight Python program designed to handle large CSV files using the pandas library.
It allows you to:

✔ Features

Load a CSV file (first.csv)

Sort the data in ascending order by any column

Search for specific values in any column

Display dataset summary (count, mean, min, max, unique values, etc.)

Export sorted results to sorted_output.csv

Clean functional structure with simple variable names

✔ Requirements

Python 3.8+

pandas library
